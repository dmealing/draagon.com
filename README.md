# draagon.com

Company website for Draagon — an AI development company building intelligent systems including voice assistants, creative AI tools, and immersive experiences.

## Stack

- Pure HTML/CSS (no build step)
- Hosted on Cloudflare Pages
- DNS managed by Cloudflare

## Development

Open `index.html` in a browser. No build tools required.

## Deployment

Pushes to `main` auto-deploy to Cloudflare Pages via GitHub integration.

## Projects Featured

- **Roxy Voice Assistant** — Agentic voice assistant with persistent memory
- **Draagon AI** — AI development platform and toolchain
- **Party Lore** — AI-powered immersive experience platform
